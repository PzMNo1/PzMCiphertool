# CipherTool 后端部署指南

## 技术栈
- **前端**：JavaScript + CSS + HTML
- **后端**：Spring Boot 3.2 + Redis
- **短信服务**：阿里云 SMS API

---


## 前端对接

修改 `frontendciphertool/loginsystem/auth.js`：

```javascript
// 后端API地址
const API_BASE = 'http://localhost:8080';

// handleSendCode 函数中
const res = await fetch(`${API_BASE}/api/auth/send-code`, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ phone: phone })
});

// handleLogin 函数中  
const res = await fetch(`${API_BASE}/api/auth/login`, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ phone: phone, code: code })
});
```

## API 接口文档

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/auth/send-code` | POST | 发送验证码 |
| `/api/auth/login` | POST | 登录验证 |
| `/api/auth/health` | GET | 健康检查 |

### 请求示例

**发送验证码：**
```json
POST /api/auth/send-code
{ "phone": "13800138000" }
```

**登录：**
```json
POST /api/auth/login
{ "phone": "13800138000", "code": "123456" }
```

## 注意事项

1. 开发环境配置已内置，可直接运行
2. **生产环境**建议通过环境变量覆盖敏感配置
3. 已内置发送频率限制（60秒/次，5次/小时）
4. 验证码5分钟自动过期
5. 支持跨域访问（开发环境允许所有来源）
